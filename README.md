# 🤖 ResuMind AI – Smart Resume Analyzer

🚀 A premium resume analysis web application that evaluates resumes, detects skills, calculates ATS score, and provides role-based suggestions to help students and job seekers improve their chances of getting selected.

## 🌟 Live Demo

👉 (https://resumind-ai-koxtuszgapswsb42y7j2fq.streamlit.app/)

---

## 📌 About the Project

ResuMind AI is designed to help users analyze their resumes in a smart and structured way. It mimics how Applicant Tracking Systems (ATS) evaluate resumes and provides actionable insights.

This tool is especially useful for:

* 🎓 Students applying for internships
* 💼 Job seekers targeting specific roles
* 🤖 Beginners entering tech/AI fields

---

## 🎯 Key Features

### 🔹 Resume Upload

* Upload PDF resumes easily
* Automatically extracts and processes text

### 🔹 Skill Detection

* Detects technical skills from resume
* Displays them as interactive badges

### 🔹 ATS Score Calculation

* Calculates resume strength based on detected skills
* Categorizes into:

  * 🔴 Weak
  * 🟡 Average
  * 🟢 Strong

### 🔹 Role-Based Skill Analysis

* Select a target role (e.g., AI Engineer, Developer)
* Compares your skills with required skills
* Identifies missing skills

### 🔹 Smart Suggestions

* Provides improvement tips based on score
* Suggests skills/technologies to learn

### 🔹 PDF Report Download

* Generate downloadable resume analysis report
* Includes score, skills, and missing areas

---

## 🛠️ Tech Stack

* **Frontend & Backend:** Streamlit
* **Programming Language:** Python
* **Libraries Used:**

  * PyPDF2 – PDF text extraction
  * ReportLab – PDF report generation
* **UI Design:** Custom CSS (Glassmorphism + Dark Theme)

---

## 🧠 How It Works

1. User uploads a resume (PDF)
2. Text is extracted using PyPDF2
3. Skills are identified using keyword matching
4. ATS score is calculated
5. Role-based comparison is performed
6. Missing skills are highlighted
7. Suggestions are generated
8. Report is available for download
